Question 2: What are CSS pseudo-elements, and how are they used? Provide examples.

Answer: CSS pseudo-elements are special keywords added to selectors that allow you to style specific parts of an element without adding extra HTML markup. can style the first letter, or line, of an element.

Example of CSS pseudo-elements

  • ::before
  • ::after
  • ::first-letter
  • ::first-line

Example -> ::first-letter and ::first-line. For the first letter I set the style for the 'P' character.

Pseudo-elements allow you to style specific parts of an element without needing to add extra HTML. This is useful for adding decorative content or special styles to elements.
